# 昊道文化 GitHub Pages Website

這是一個可直接上傳到 GitHub 的靜態動態網站原型，不需要安裝套件或建置流程。

## 檔案

- `index.html`：網站內容與架構
- `styles.css`：視覺風格與響應式版面
- `script.js`：圖片輪播、首頁書法字流淌、淡入、分頁、FAQ、彈窗與手機選單
- `assets/`：網站使用的 SVG 視覺素材

## GitHub Pages 使用方式

1. 建立一個 GitHub repository。
2. 上傳本資料夾內所有檔案到 repository 根目錄。
3. 到 `Settings > Pages`。
4. Source 選擇 `Deploy from a branch`。
5. Branch 選擇 `main` 與 `/root`。
6. 儲存後等待 GitHub Pages 產生網址。

## 動態效果

- 首頁首圖輪播
- 書法字流淌 canvas 動畫
- 滾動淡入
- 卡片 hover
- 活動與作品詳情彈窗
- 三大法舟 tab 切換
- FAQ 展開收合
- 手機選單
